# dxa-compiler

Future home of the DEXA reference compiler (Rust): lexer, parser, typechecker, IR, and interpreter.

Currently developed in [dxa-dev](https://github.com/dxiv/dxa-dev) alongside the website. This repo is prepared for a possible split so the compiler can be versioned and released independently.

- **Language:** DEXA (`.dxa` files)
- **CLI:** `dxc`
- **Output:** DX-IR, execution trace, gas
